ISA2OWL

Package to convert ISA-TAB files into RDF data using different semantic frameworks (OWL ontologies).

Download this project as a .zip file Download this project as a tar.gz file

Overview

ISA2OWL is a Java package relying on the OWLAPI that parses ISA-TAB files and generates an RDF representation, relying on mapping files from the ISA syntax to different semantic frameworks (or OWL ontologies).
